# Tic-Tac-Toe Game

This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript.
I created this project to practice JavaScript logic, DOM manipulation,
and basic front-end development concepts.

## Features
- Two-player Tic-Tac-Toe game (X and O)
- Shows winner message
- Reset button to restart the game
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
- index.html
- style.css
- script.js

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. Start playing the game

## Purpose of This Project
This is a beginner-level project created to practice the basics of
HTML, CSS, and JavaScript.
This project helped me strengthen my foundation in front-end development.
